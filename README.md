# PollingDapp : vote and get paid

## Installation

### Deploy and run node on localhost:

Terminal n°1:

```shell
cd pollingsDapp
npx hardhat node
```

Terminal n°2:

```shell
cd pollingsDapp
npx hardhat run --network localhost scripts/deploy.ts
```

### Run frontend app

Terminal n°3

```shell
cd pollingsDapp/app
npm start
```
